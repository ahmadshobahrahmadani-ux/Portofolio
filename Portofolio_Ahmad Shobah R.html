Company Profile Website Deployment on AWS EC2 Cloud

Melakukan deployment website company profile pada Amazon EC2 menggunakan public IP sehingga website dapat diakses secara online melalui internet tanpa menggunakan layanan hosting pihak ketiga.

  • Melakukan provisioning dan konfigurasi instance Amazon EC2 pada AWS Academy Sandbox.
  • Menginstal dan mengonfigurasi web server (Apache/Nginx) pada sistem operasi Linux.
  • Mengembangkan website company profile yang terdiri dari halaman Home, About, Services/Product, dan Contact.
  • Mengimplementasikan navigasi antar halaman menggunakan HTML, CSS, dan JavaScript.
  • Mengonfigurasi security group agar website dapat diakses melalui HTTP dari jaringan publik.
  • Melakukan deployment menggunakan Public IP EC2 dan menguji aksesibilitas layanan melalui browser.

Environment Monitoring Device untuk Greenhouse Urban Farming Wokwi Simulator

Aplikasi monitoring lingkungan berbasis Arduino Uno untuk memantau kondisi greenhouse secara real-time melalui pengukuran suhu, kelembapan udara, kualitas udara, kelembapan tanah, dan level air irigasi.

  • Merancang sistem monitoring menggunakan sensor DHT22, MQ-135, HC-SR04, dan Soil Moisture Sensor.
  • Mengembangkan logika klasifikasi kondisi menjadi Normal, Peringatan, dan Bahaya berdasarkan nilai ambang (threshold) setiap sensor.
  • Mengimplementasikan tampilan data sensor secara bergilir (scrolling) pada LCD 16×4.
  • Menambahkan indikator visual berupa LED berkedip saat kondisi memasuki status bahaya.
  • Menyusun kode secara modular untuk pembacaan sensor, evaluasi status, dan pengelolaan tampilan.

Sistem Lampu Otomatis dengan Monitoring Suhu Ruangan Wokwi Simulator

Sistem otomasi ruangan sederhana yang mengontrol pencahayaan berdasarkan intensitas cahaya lingkungan serta menampilkan suhu ruangan secara real-time pada LCD.

  • Merancang rangkaian berbasis sensor cahaya (LDR), sensor suhu NTC, LED, dan LCD 16×2.
  • Mengembangkan logika kontrol otomatis untuk menyalakan dan mematikan lampu berdasarkan nilai intensitas cahaya.
  • Mengintegrasikan pembacaan suhu ruangan ke dalam tampilan LCD secara real-time.
  • Mengimplementasikan pemrosesan data sensor analog untuk kebutuhan monitoring dan kontrol.
  • Mendokumentasikan struktur program agar mudah dipahami dan dikembangkan lebih lanjut.

Animal Eat & Avoid — Educational Java Game Java & Greenfoot

Game edukasi berbasis Java menggunakan Greenfoot yang mengajarkan pola makan hewan melalui mekanisme menangkap makanan yang sesuai dan menghindari rintangan. Permainan terdiri dari 3 level dengan sistem skor, nyawa, timer, dan tingkat kesulitan yang meningkat.

  • Mengimplementasikan konsep Object-Oriented Programming (OOP) menggunakan Java.
  • Mengembangkan sistem skor, nyawa (lives), countdown timer, dan level progression.
  • Menerapkan collision detection untuk interaksi karakter, makanan, dan rintangan.
  • Merancang antarmuka permainan serta mekanisme game over dan kemenangan.
  • Melakukan pengujian terhadap 10 pengguna dan mengevaluasi umpan balik untuk perbaikan gameplay.
